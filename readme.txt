== Description ==

This plugin Allow User To Skip His Payment to Next Period

Feature Can Be Enabled From General Setting.

== Installation ==
1. Upload 'wc-sub-skip-shipment.php' to the '/wp-content/plugins/' directory,
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enable Feature From Woocommerce > Settings > WC Sub Box Extra Action
